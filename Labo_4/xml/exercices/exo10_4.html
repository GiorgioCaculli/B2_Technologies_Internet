Les nuages

1. ALTOCUMULUS

Ce type de nuage possède les espèces suivantes :

  • lenticularis
  • stratiformis
  • castellanus
  • flocus

2. CIRRUS

Ce type de nuage possède les espèces suivantes :

  • flocus
  • castellanus
  • spissatus
  • uncinus
  • fibratus

3. CIRROCUMULUS

Ce type de nuage possède les espèces suivantes :

  • lenticularis
  • stratiformis
  • flocus
  • castellanus

4. CUMULUS

Ce type de nuage possède les espèces suivantes :

  • fractus
  • humilis
  • mediocris
  • congestus

5. STRATO-CUMULUS

Ce type de nuage possède les espèces suivantes :

  • stratiformis
  • lenticularis
  • castellanus

6. CUMULONIMBUS

Ce type de nuage possède les espèces suivantes :

  • calvus
  • capillatus
  • mammatus
