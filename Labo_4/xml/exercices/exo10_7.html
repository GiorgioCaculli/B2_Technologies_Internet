Les nuages

1. Altocumulus

Ce type de nuage possède les 4 espèces suivantes :

  • lenticularis
  • stratiformis
  • castellanus
  • flocus

2. Cirrus

Ce type de nuage possède les 5 espèces suivantes :

  • flocus
  • castellanus
  • spissatus
  • uncinus
  • fibratus

3. Cirrocumulus

Ce type de nuage possède les 4 espèces suivantes :

  • lenticularis
  • stratiformis
  • flocus
  • castellanus

4. Cumulus

Ce type de nuage possède les 4 espèces suivantes :

  • fractus
  • humilis
  • mediocris
  • congestus

5. Strato-cumulus

Ce type de nuage possède les 3 espèces suivantes :

  • stratiformis
  • lenticularis
  • castellanus

6. Cumulonimbus

Ce type de nuage possède les 3 espèces suivantes :

  • calvus
  • capillatus
  • mammatus
